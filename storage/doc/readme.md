## 文档书写要求

1. 生成文档

   需要事先在storage/doc目录中创建好文档，同时在storage/doc/config.json中按要求配置

2. 语法

   请严格参照markdown语法来书写

3. 条目

   * 接口名称(必须)
   * 接口地址(必须)
   * 请求方式(必须) GET/POST/DELETE/PUT/PATCH/OPTIONS
   * 请求参数(必须) 必填参数在前，非必填在后
   * 响应结果(必须) json格式
   * 错误示例(非必须)
